## About project
Nexus Portal is an interactive library designed to guide developers through the vast world of IT. It categorizes programming languages and career paths (Frontend, Backend, GameDev) into "Factions", making the learning process feel like an RPG game.

## ✨ Key Features
- Gamified Experience: "Choose your side" mechanic for selecting tech stacks.
- Modern UI/UX: Dark mode aesthetics with glassmorphism and neon accents.
- Interactive Components: Custom tabs, modals, and smooth transitions.

## 🛠️ Tools & Tech Stack
- Core: Next.js, TypeScript
- Styling: Tailwind CSS
- Animations: Framer Motion
